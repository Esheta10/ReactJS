# ReactJS Project

This repository serves as a workspace for my ReactJS learning journey, containing various practice projects, components, and exercises.

## Repository Structure

The project is organized into modular directories to focus on specific React concepts:

*   **App-1 & App-2**: Functional application prototypes and experiments.
*   **Hooks**: Practice exercises implementing React Hooks such as `useState`, `useEffect`, and custom hooks.
*   **Props**: Examples focused on parent-to-child component communication and data passing.
*   **React-Router**: Implementations of navigation and client-side routing.
*   **Context API**: Practice implementations for managing global state and avoiding prop drilling.
